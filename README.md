# Hermes Agent AI Workflow Demo

This repository demonstrates a complete AI-assisted development workflow using Hermes Agent.

## Project Purpose

The purpose of this project is to provide evidence of my practical ability to use AI Agent tools to complete real development tasks, including task planning, code generation, debugging, terminal execution, and final result output.

## Tools Used

- Hermes Agent
- Python
- Terminal / PowerShell
- GitHub

## Workflow Overview

The project follows this workflow:

1. Define the development task
2. Use Hermes Agent to break down the task
3. Generate Python code with Agent assistance
4. Run the code in terminal
5. Capture terminal errors
6. Use Hermes Agent to debug and fix the code
7. Run the corrected code successfully
8. Save the final output
